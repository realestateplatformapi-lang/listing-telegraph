# Объявление → Telegraph

Сервис для преобразования объявлений OLX.ua и Rieltor.ua (включая страницы агентств на поддоменах) в статью Telegraph.

```sh
python3 -m pip install -r requirements.txt
python3 app.py
```

Откройте http://localhost:8080. Для постоянного Telegraph-аккаунта задайте переменную окружения `TELEGRAPH_ACCESS_TOKEN`.

В Finder можно просто дважды нажать `start.command`. Если macOS покажет предупреждение,
нажмите по файлу правой кнопкой → «Открыть». Также он запускается из Terminal командой
`./start.command`.

Сервис получает публичные Open Graph/JSON-LD данные. Если сайт показывает антибот-проверку, данные извлечь не удастся.

## Публичный доступ для других людей

Проект готов к запуску как Docker-сервис на Railway, Render, Fly.io или собственном сервере. Подключите репозиторий/загрузите эти файлы, выберите **Dockerfile** как способ сборки и задайте переменную `TELEGRAPH_ACCESS_TOKEN` (необязательно, но рекомендовано). Хостинг сам передаст порт через `PORT`; сервер слушает внешний интерфейс `0.0.0.0`.

Для запуска на сервере с Docker:

```sh
docker build -t listing-telegraph .
docker run -d --restart unless-stopped -p 8080:8080 --name listing-telegraph listing-telegraph
```

Чтобы сделать адрес доступным по HTTPS, направьте домен на сервер через обратный прокси (например, Caddy или Nginx). Для настоящего публичного запуска также желательно установить постоянный `TELEGRAPH_ACCESS_TOKEN` в переменных окружения хостинга.
