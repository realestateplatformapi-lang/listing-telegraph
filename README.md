# Объявление → Telegraph

Сервис для преобразования объявлений OLX.ua и Rieltor.ua (включая страницы агентств на поддоменах) в статью Telegraph.

```sh
python3 -m pip install -r requirements.txt
python3 app.py
```

Откройте http://localhost:8080. Для постоянного Telegraph-аккаунта задайте переменную окружения `TELEGRAPH_ACCESS_TOKEN`.

В Finder можно просто дважды нажать `start.command`. Если macOS покажет предупреждение,
нажмите по файлу правой кнопкой → «Открыть». Также он запускается из Terminal командой
`./start.command`.

Сервис получает публичные Open Graph/JSON-LD данные. Если сайт показывает антибот-проверку, данные извлечь не удастся.

## KYIV ESTATE

Актуальний інтерфейс створює узгоджені українську й англійську версії, показує ціни в UAH/USD/EUR за курсом НБУ, зберігає локальні пакети та manifest, а також підтримує PDF і двомовну публікацію в Telegraph.

На Windows усі вибрані фото спочатку зберігаються в `DATA_ROOT/listings/<id>/original`, а дозволені до публікації версії — у `final`. Якщо налаштовано `KYIV_ESTATE_AI_ENDPOINT`, застосунок передає одне оголошення на наявний локальний AI-конвеєр, очікує сертифіковані фото й лише після цього створює пакет. Перед публікацією фінальні файли та логотип завантажуються у сховище Telegraph і кешуються за SHA-256, тому готова сторінка не залежить від вихідного сайту або тимчасового Windows-тунелю.

Логотип обов’язково розміщується другим зображенням — одразу після головного фото. Джерелом логотипа може бути постійний `KYIV_ESTATE_LOGO_PATH` на Windows або HTTPS `KYIV_ESTATE_LOGO_URL` на сервері.

Інструкції для Windows і синхронізації з Mac знаходяться в каталозі `windows/`. Почніть із `windows/README_WINDOWS.md` або повної інструкції, підготовленої разом із проєктом.

## Публичный доступ для других людей

Проект готов к запуску как Docker-сервис на Railway, Render, Fly.io или собственном сервере. Подключите репозиторий/загрузите эти файлы, выберите **Dockerfile** как способ сборки и задайте переменную `TELEGRAPH_ACCESS_TOKEN` (необязательно, но рекомендовано). Хостинг сам передаст порт через `PORT`; сервер слушает внешний интерфейс `0.0.0.0`.

Для запуска на сервере с Docker:

```sh
docker build -t listing-telegraph .
docker run -d --restart unless-stopped -p 8080:8080 --name listing-telegraph listing-telegraph
```

Чтобы сделать адрес доступным по HTTPS, направьте домен на сервер через обратный прокси (например, Caddy или Nginx). Для настоящего публичного запуска также желательно установить постоянный `TELEGRAPH_ACCESS_TOKEN` в переменных окружения хостинга.

Основні змінні середовища:

- `DATA_ROOT` — постійне сховище пакетів, SQLite, оригіналів і фінальних фото;
- `KYIV_ESTATE_LOGO_PATH` або `KYIV_ESTATE_LOGO_URL` — фірмовий логотип;
- `KYIV_ESTATE_PHONE`, `KYIV_ESTATE_URL`, `KYIV_ESTATE_INSTAGRAM`, `KYIV_ESTATE_TELEGRAM`, `KYIV_ESTATE_WHATSAPP`, `KYIV_ESTATE_FACEBOOK`, `KYIV_ESTATE_EMAIL` — клікабельні контакти;
- `KYIV_ESTATE_AI_ENDPOINT` — локальний endpoint наявного Windows AI-конвеєра;
- `KYIV_ESTATE_AI_PACKAGES_ROOT` — його каталог сертифікованих пакетів на D;
- `KYIV_ESTATE_SOURCE_LISTINGS_ROOT` — архів оголошень Блоку 2 для відновлення оригіналів, коли вихідні CDN-посилання вже протерміновані;
- `KYIV_ESTATE_AI_REQUIRED=true` — заборонити fallback на оригінали, якщо AI-перевірка недоступна;
- `KYIV_ESTATE_MAX_PHOTOS` — кількість фото оголошення, від 1 до 100.

## Два режими фотографій

- **Browser** швидко читає оголошення і залишає тільки медіафайли самого offer; логотипи сайту, меню, аватари, платіжні іконки та емодзі відсіюються до показу.
- **AI processing** очікує Windows-конвеєр, прибирає людей, сторонні агентські зображення й водяні знаки, а потім показує паралельно `AI processed · Main` та `Original`.
- Золота колонка є основною. Фотографію можна перетягнути між колонками, щоб вибрати оброблену або оригінальну версію; прапорець визначає включення до Telegraph/PDF.
- Редактор показує `Original text` окремо від редагованого `AI text · Main`, а ціни й характеристики винесені у блоки з емодзі.
